# LSPR 公式白话版（非 LaTeX）

本文把论文版中的公式改写成“普通可读数学表达”，不使用 LaTeX 语法。

## 符号说明

- λ：波长
- c：浓度（ng/ml）
- z：对数浓度
- s(λ)：输入光谱
- ŝ(λ)：预测光谱
- A：峰强
- λ0：峰位
- FWHM：半高全宽
- γ：线宽参数，等于 FWHM/2

---

## 公式(1) Lorentz 峰形重建

I(λ) = A × [γ² / ((λ - λ0)² + γ²)]

其中：γ = FWHM / 2

解释：
- 峰位由 λ0 决定
- 峰高由 A 决定
- 峰宽由 FWHM 决定

---

## 公式(2) 残差网络预测峰特征变化

输入特征 = [log10(c + 0.001), λ_pre, A_pre, FWHM_pre]

网络输出 = [Δλ, ΔA]

解释：
- Δλ 表示峰位偏移
- ΔA 表示峰强变化

---

## 公式(3) 后态特征重建

λ_post = λ_pre + Δλ

A_post = A_pre + ΔA

---

## 公式(4) v1 浓度预测

z_hat_v1 = Predictor_v1( MinMaxNormalize(光谱) )

解释：
- 先对单条光谱做最小值-最大值归一化
- 再送入 v1 网络，输出对数浓度

---

## 公式(5) 生成器反归一化

s_hat_norm = Generator(z_hat)

s_hat = s_hat_norm × (spec_max - spec_min) + spec_min

解释：
- 网络先生成 0~1 归一化光谱
- 再用训练时保存的全局范围还原到真实强度

---

## 公式(6) 生成器训练损失

L_gen = L_MSE + α × L_shift

其中：
- L_MSE = 均方误差(预测光谱, 真实光谱)
- L_shift = 均方误差(COM(预测光谱), COM(真实光谱))
- COM = 光谱质心（中心位置）

---

## 公式(7) v2 双通道稳健标准化

通道1（原始强度）：
X1 = (s - median(s)) / (IQR(s) + ε)

通道2（一阶导数）：
X2 = (gradient(s) - median(gradient(s))) / (IQR(gradient(s)) + ε)

解释：
- median 是中位数
- IQR 是四分位距（Q75-Q25）
- 比普通标准化更抗异常值

---

## 公式(8) v2 浓度预测

z_hat_v2 = Predictor_v2([X1, X2])

解释：
- 同时看强度与谱形变化趋势（导数）

---

## 公式(9) 单调惩罚

把样本按真实浓度从低到高排序后：

L_mono = 平均值( max(0, z_hat_i - z_hat_(i+1)) )

解释：
- 如果低浓度样本预测值比高浓度还大，就会被惩罚
- 作用是让预测随浓度更单调

---

## 公式(10) v2 总损失

L_v2 = L_Huber + β × L_mono

解释：
- Huber 负责主回归
- 单调惩罚负责排序一致性

---

## 公式(11) isotonic 校准

z_hat_cal = f_isotonic(z_hat_v2)

c_hat = 10^(z_hat_cal) - 0.001

解释：
- 先做单调映射纠偏
- 再从对数域转回浓度域

---

## 公式(12) 强度对齐（scale + offset）

s_hat_aligned = a × s_hat + b

解释：
- a 调整动态范围（拉伸/压缩）
- b 调整基线（整体上移/下移）

---

## 公式(13) a 和 b 的计算（分位数法）

a = [Q90(ref) - Q10(ref)] / [Q90(pred) - Q10(pred) + ε]

b = Q50(ref) - a × Q50(pred)

其中：
- ref = 参考光谱（通常是输入光谱或物理曲线）
- pred = 预测光谱
- Q10/Q50/Q90 = 10%/50%/90%分位值

解释：
- 不是“只对齐峰值”
- 是按整体强度分布做稳健对齐

---

## 一句话总结

当前系统的公式链路可以理解为：

“先用深度学习反演浓度（v2+单调校准），再由生成器出光谱，最后做强度分布对齐”，
从而同时提升浓度精度和峰高可比性。
